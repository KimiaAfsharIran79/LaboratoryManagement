\# Laboratory Management System



A desktop Laboratory Management System developed with C# and Windows Forms using a layered architecture.



\## Overview



This project was developed as an educational project to manage laboratory operations such as patients, employees, laboratory tests, units, insurance types, and reception records.



The application follows a multi-layer architecture and uses SQL Server for data storage.



\## Features



\- User Login

\- Patient Management

\- Employee Management

\- Laboratory Test Management

\- Test Category Management

\- Test Range Management

\- Unit Management

\- Insurance Type Management

\- Reception Registration

\- Reports



\## Technologies



\- C#

\- Windows Forms (.NET Framework)

\- SQL Server

\- ADO.NET

\- Layered Architecture



\## Project Structure



```

Laboratory

│

├── DataAccess

├── DomainModel

├── Framework

└── Laboratory (UI)

```
